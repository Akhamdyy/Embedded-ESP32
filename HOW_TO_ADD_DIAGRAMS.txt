2. Create new diagram
3. Add 7 circles for states

5. Label arrows with conditions

7. Export as PNG or PDF
8. Copy-paste image into Word document

HOW TO INSERT DIAGRAM INTO WORD:

1. Open PHASE1_DESIGN.docx in Microsoft Word
2. Go to the section that says "Create in draw.io..."
3. Delete that line
4. Click Insert → Pictures → This Device
5. Select your diagram PNG/PDF file
6. Click Insert
7. Resize diagram to fit page
8. Save

═══════════════════════════════════════════════════════════════

DRAW.IO QUICK START:

Visit: https://draw.io (no login needed)

1. Click "Create New Diagram"
2. Choose "Blank Diagram"
3. Left panel: Drag shapes (rectangle, circle, etc.)
4. Connect with arrows
5. Add text labels
6. File → Export As → PNG (for Word)
7. Copy/Paste into Word
